# 测试环境最小联调检查清单

## 一、基础联通

- 后台首页可打开
- 后台登录页可打开
- `GET /api/health` 返回成功
- `/storage/...` 资源可访问

## 二、后台登录

- 使用测试管理员账号可登录
- 登录后可进入模板管理
- 登录后可进入订单管理

## 三、模板链路

- 后台可查看模板列表
- 后台可查看模板详情
- 后台推荐模板后，用户端首页推荐位有变化
- 后台下架模板后，用户端模板入口消失

## 四、作品链路

- 用户端可以保存作品到后端
- 我的作品列表可读到后端数据
- 历史作品回看不会被新模板版本影响

## 五、图片资产

- 上传图片后，作品图片不再只依赖设备本地路径
- 后端返回的图片地址在测试环境下可直接访问
- 后台查看作品时能看到真实预览图

## 六、地址链路

- 地址列表可读后端数据
- 新增地址成功
- 编辑地址成功
- 删除地址成功
- 默认地址能在确认订单页自动带出

## 七、订单链路

- 确认订单页读取后端作品详情
- 确认订单页读取后端地址数据
- 能成功创建后端订单
- 我的订单列表优先读取后端数据
- 订单详情页能看到冻结快照

## 八、动作占位链路

- 后台订单详情可看到 支付 / PDF / 履约 3 个动作入口
- 被拦截时能看到明确原因
- 可进入时只显示占位说明
- 点击动作不会改变订单状态

## 九、当前不在测试通过范围内

- 真支付
- 支付回调
- PDF 真实生成
- 发货 / 物流
- 退款售后
