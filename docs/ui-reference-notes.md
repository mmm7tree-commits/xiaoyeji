# UI 参考规范提炼

参考链接：https://www.zcool.com.cn/work/ZMzgzMDQzMTI=.html

参考作品：儿童教育类 app_小布壳阅读_作品集 part2。

## 可落地到小页记的视觉规则

- 主色：清新绿色，参考色 `#31A381`、`#4DC085`、`#40765A`。
- 背景：大面积浅灰白和留白，参考色 `#F5F6F5`、`#F1F3F6`。
- 点缀：少量暖橙/珊瑚用于强调，不作为主操作色。
- 卡片：白底、圆角、轻阴影，避免厚重装饰。
- 信息层级：先讲核心价值，再给操作入口，列表信息保持干净。
- 儿童感：用绘本、故事、陪伴、成长这类语义，不使用过度电商化表达。

## 本轮应用范围

- 全局导航、Tab、页面背景、按钮、卡片、标签。
- 首页首屏和制作流程文案。
- 模板卡片、预览页参考图、作品集、订单、我的、地址、后台的主色和文本层级。
